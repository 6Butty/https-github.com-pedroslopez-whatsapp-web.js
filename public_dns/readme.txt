Introduction
------------
Public DNS Server Tool is a small utility for changing the DNS servers in Windows XP, Vista, 7, 8, 8.1 and 10 on the fly. You can use the Public DNS Server Tool to quickly switch the DNS servers to one of the well known public DNS servers. You can also add your own DNS servers by editing the servers.ini file.


Requirements
------------
Windows XP/Vista/7/8/8.1/10
It was tested on Windows 10 32-bit/64-bit, Windows 7 32-bit and Windows 8 64-bit only. I do not have other Windows versions anymore.


Install/Uninstall
------------------
Not required. It is a portable application. Just unzip to a folder and run the program.
